# void57-chess Library Test

A React + TypeScript test application for the [void57-chess](https://www.npmjs.com/package/void57-chess) chess library.

## About

This project serves as a testing and demonstration platform for the `void57-chess` npm package, showcasing both standard chess and Chess960 gameplay capabilities.

## Features

- **Standard Chess**: Play regular chess with full rule validation
- **Chess960**: Fischer Random Chess with all 960 starting positions
- **Custom Positions**: Enter specific Chess960 position numbers (0-959)
- **Pawn Promotion**: Choose promotion piece (Queen, Rook, Bishop, Knight)
- **Move History**: Track and undo moves
- **FEN Support**: Copy FEN strings and position data
- **Responsive Design**: Optimized for desktop, tablet, and mobile
- **Dark Theme**: Wooden board aesthetic with comfortable colors

## Chess Library

This application uses the **void57-chess** library:

- **NPM Package**: [void57-chess](https://www.npmjs.com/package/void57-chess)
- **Version**: 1.0.1
- **Features**: Full chess rules, Chess960 support, move validation, game state management
